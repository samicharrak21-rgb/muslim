# الديوان — النسخة النهائية الكاملة الشاملة

هذه الحزمة تحتوي على:
- تطبيق Android-ready بدون مفاتيح API خارجية.
- موقع ويب كامل بنفس الواجهة داخل مجلد `../diwan-site`.
- القرآن كاملاً محلياً.
- مواقيت وقبلة محلياً عبر adhan-js.
- تذكيرات محلية قابلة للربط دون خادم.
- شاشة بداية وشعار مخصص.

## لماذا بدون مفاتيح؟
- لا توجد خرائط Google.
- لا توجد خدمات سحابية إلزامية.
- المواقيت والقبلة تعمل محلياً.
- البحث في القرآن محلي.

## تشغيل الموقع
افتح ملف `diwan-site/diwan-site.html` أو ارفعه كملفات static.

## تشغيل التطبيق
- استخدم الحزمة الأساسية داخل هذا المشروع.
- ابنِ debug APK عبر Android/Gradle عند الحاجة.
