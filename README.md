# الديوان - النسخة المحلية النهائية

مشروع محلي نهائي داخل ZIP جاهز للمعاينة والرفع لاحقاً.

## الميزات
- واجهة أندلسية عربية.
- 114 ملف سورة منظمة.
- أذكار وأدعية.
- مواقيت صلاة وقبلة عبر adhan.
- صوتيات أولية لقراء معروفين.
- بناء ويب محلي وتهيئة Android عبر Capacitor.

## الأوامر
npm install
npm run doctor
npm run build
npm run prepare:android

## ملاحظة
ملفات السور الحالية منظمة وجاهزة للاستبدال بالنص الكامل عبر دليل docs/HYDRATE-QURAN.md.


## تحديث جديد
- تم إدخال النص الكامل للقرآن داخل ملفات السور محلياً اعتماداً على quran-json dataset.


## النسخة الكاملة الآن
- نص القرآن الكامل داخل المشروع.
- بحث داخلي في الآيات.
- تشغيل صوتي تجريبي لأول آية من السورة المختارة.
- أوقات سنة الليل متاحة في منطق التطبيق عبر adhan.
